# Paymebear


## Dependencies
- Docker
- Docker Compose


## File Structure

```
    .
    └── webclient
            ├── constants
            ├── constructors
            ├── components
            │       ├── App.js - client application root component
            │       └── ...
            │
            ├── context - React context
            ├── contracts - Ethereum contract abi's
            ├── constructors
            ├── pages
            │     ├── _app.js
            │     ├── _document.js
            │     ├── ...
            │     └── index.js
            │
            ├── public
            ├── server - NextJS webserver
            ├── styles
            ├── tests
            │     ├── integration
            │     └── unit
            │
            └── utils

```


## Getting Started

```bash
cd webclient && yarn dev
```
